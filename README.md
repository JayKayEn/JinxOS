# JinxOS

## About

JinxOS is a toy *nix-style operating system kernel inspired by MIT's JOS and Harvard's OS/161.

## Features:

* Kernel threads
* Virtual memory
* Malloc/free
* Userspace programs
* Interrupts/syscalls
* Thread-safe data structures
* Simple test suite
